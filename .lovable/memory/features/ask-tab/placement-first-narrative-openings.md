name: Placement-First Narrative Openings
description: Every narrative_section body must open with specific chart placements (sign+house+ruler), never abstract statements
type: preference
---
Every narrative body paragraph must follow this formula:
1. First sentence: name the key placements by sign + house (and ruler if relevant)
2. Second sentence: translate what that specific configuration means for THIS person
3. Then continue into broader patterns

FORBIDDEN: Opening with abstract statements like "Your relationship blueprint is built on a contradiction" or "This year brings transformation."

Applied in:
- SYSTEM_PROMPT in ask-astrology edge function (global rule)
- AskQuickTopics.tsx relationship prompt (section-specific reinforcement for Sections 1-3)
